Bases de datos

Las bases de datos son conjuntos de información que nos ayudan a que podamos guardar información normalmente en una tabla. Existen diferentes tipos de bases de datos en el mercado como bases de datos relacionales, no relaciones, de red,etc. Existen diferentes empresas que nos ofrecen sus servicios para que nosotros podamos guardar información algunos de los más populares son: Oracle, Microsoft, IBM, entre otros.

Funciones

Las funciones nos ayudan a que podamos realizar diferentes tipos de operaciones matemáticas, lógicas,religiosas, ,entretenimiento,etc. En las bases de datos podemos encontrar normalemente los siguientes tipos de funciones como lo son:
  • Funciones del sistema o predifinidas
  • Funciones definidas por el usuarios

    • Dentro de estas funciones definidas podemos encontrar como

    Funciones escalares:Devuelven un solo valor

    Funciones de valor tabla en línea:La cual solo puede tener instrucciones Trans SQL

    Funciones de valor de míñtiples tablas: Contienen varias instrucciones de Trans-SQL y regresa una tabla
